===============================================================
USAGE INSTRUCTIONS
===============================================================

FRONTEND (TypeScript/Next.js):

1. Chat Endpoint:
   POST /api/chat
   Headers: Authorization: Bearer <token>
   Body: {
     "messages": [{"role": "user", "content": "..."}, ...],
     "patientData": {"age": 45, "gender": "M", ...},
     "specialty": "Cardiology"
   }

2. Transcribe Endpoint:
   POST /api/transcribe
   Headers: Authorization: Bearer <token>
   Body: FormData with "audio" file

3. Auth Verification:
   POST /api/auth/verify
   Headers: Authorization: Bearer <token>
   Body: {"token": "<firebase-token>"}

4. Patients CRUD:
   GET /api/patients - List all
   POST /api/patients - Create new
   Headers: Authorization: Bearer <token>
   Body: {
     "name": "...",
     "email": "...",
     "dateOfBirth": "YYYY-MM-DD",
     "gender": "M/F"
   }

5. PDF Generation:
   POST /api/pdf
   Headers: Authorization: Bearer <token>
   Body: {
     "patientName": "...",
     "patientAge": 45,
     "patientGender": "M",
     "specialty": "Cardiology",
     "subjective": "...",
     "objective": "...",
     "assessment": "...",
     "plan": "..."
   }

PYTHON MICROSERVICE:

1. Start server:
   cd python-services
   pip install -r requirements.txt
   python main.py

2. Health check:
   GET http://localhost:8000/health

3. Causal analysis:
   POST http://localhost:8000/analyze
   Body: {
     "patientData": {...},
     "variables": ["var1", "var2", ...],
     "outcome": "target_variable",
     "confounders": ["conf1", ...]
   }

4. API documentation:
   http://localhost:8000/docs
