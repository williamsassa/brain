🔑 CONFIGURATION - API KEYS REQUIRED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. FIREBASE (Google Authentication)
   Website:    console.firebase.google.com
   Steps:
   ☐ Create new project "brain-health"
   ☐ Enable Authentication service
   ☐ Add Google OAuth provider
   ☐ Add your domain to Authorized domains list
   ☐ Create Service Account and download private key

   Required Keys in .env.local:
   - NEXT_PUBLIC_FIREBASE_API_KEY
   - NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN
   - NEXT_PUBLIC_FIREBASE_PROJECT_ID
   - NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET
   - NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID
   - NEXT_PUBLIC_FIREBASE_APP_ID
   - FIREBASE_ADMIN_PRIVATE_KEY
   - FIREBASE_ADMIN_CLIENT_EMAIL
   - FIREBASE_ADMIN_PROJECT_ID

2. SUPABASE (PostgreSQL Database)
   Website:    supabase.com
   Steps:
   ☐ Create new project "brain-health-db"
   ☐ Go to SQL Editor
   ☐ Run migration from: supabase/migrations/001_initial.sql
   ☐ Enable Row Level Security (RLS)

   Required Keys in .env.local:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - SUPABASE_SERVICE_ROLE_KEY
   - DATABASE_URL

3. ANTHROPIC (Claude API)
   Website:    console.anthropic.com
   Steps:
   ☐ Create API key
   ☐ Model used: claude-sonnet-4-20250514

   Required Key in .env.local:
   - ANTHROPIC_API_KEY

4. OPENAI (Whisper - Audio Transcription)
   Website:    platform.openai.com
   Steps:
   ☐ Create API key
   ☐ Model used: whisper-1

   Required Key in .env.local:
   - OPENAI_API_KEY

5. RAILWAY (Optional - For Production Deployment)
   Website:    railway.app
   Services:
   ☐ PostgreSQL database (or link Supabase)
   ☐ Python FastAPI microservice

   Required:
   - CAUSAL_ENGINE_URL

6. VERCEL (Frontend Deployment)
   Website:    vercel.com
   Steps:
   ☐ Connect GitHub repository
   ☐ Import project
   ☐ Add all environment variables from .env.local
   ☐ Deploy on push to main branch
